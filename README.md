🕵️‍♂️ Cybersecurity Escape Room 🔐
An interactive, multi-stage puzzle game built using React.js that helps users learn cybersecurity concepts in a fun, gamified way. Perfect for learners, educators, and CTF-style challenges!

🚀 Features
- 🔢 Multi-stage puzzles with increasing difficulty
- ⏱ 60-second timer per puzzle
- 💡 Hints for each question
- ✅ Real-time answer validation with feedback
- 🏆 Stage completion badges
- 🧠 Topics include: Base64 decoding, Caesar cipher, phishing detection, password strength, HTTP status codes, and more
- 🎉 Animated "escape" screen upon completion

🎮 How to Play
- Each stage has a series of cybersecurity puzzles.
- Solve each one within 60 seconds.
- Use hints if you're stuck (but try your best first 😉).
- Complete all puzzles to successfully escape!

🛠 Tech Stack
- Frontend: React.js, HTML5, CSS3
- Logic: React Hooks (useState, useEffect)
- UI/UX: Responsive design, themed backgrounds, feedback system

🔧 Installation
# Navigate to the project directory
cd cybersecurity-escape-room

# Install dependencies
npm install

# Start the development server
npm start
